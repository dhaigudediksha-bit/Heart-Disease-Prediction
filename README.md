## ❤️ Heart Disease Prediction

### 📌 Project Overview

This project is a Machine Learning web application that predicts whether a person is at risk of heart disease based on medical parameters. The application is built using Python, Scikit-learn, and Streamlit.

### 🚀 Features

- User-friendly Streamlit interface
- Predicts heart disease risk
- Uses a trained Machine Learning model
- Fast and easy to use
- Displays prediction instantly

### 📊 Dataset Features

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Maximum Heart Rate
- Exercise-Induced Angina
- Oldpeak
- ST Slope

### 🛠 Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Joblib

### ▶️ How to Run

1. Clone the repository:
   
   git clone https://github.com/dhaigudediksha-bit/Heart-Disease-Prediction.git

2. Install the required packages:
   
   pip install -r requirements.txt

3. Run the Streamlit application:
   
   streamlit run app.py
